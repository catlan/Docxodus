Booting WASM…

Click a paragraph or heading to edit it. Select text and use the ribbon (or Ctrl+B/I/U) to format. Enter splits a block, Backspace at the start merges with the previous one; Ctrl+Z undoes. Edits re-render only the affected block. Table cell text is editable (structure is fixed — Enter/Tab inside a cell are inert). Save downloads a lossless .docx.
